## Backend

How to run

Prerequisites
- MongoDB
- Nodejs

### 1. Clone repository
```sh
git clone https://github.com/dna-repo/dna-asli-backend.git
cd dna-asli-backend
```

### 2. Install dependencies
```sh
npm install
```

### 3. Configure environment variables
```sh
cp .env-example .env
# update .env value
```

### 3. Run server
```sh
npm run dev
```
